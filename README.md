# Cost-Constrained ML Screening of the Full Transition Metal Composition Space for HER Electrocatalysts

## Overview
Using a surrogate model trained on DFT adsorption energies, a broad sample of 5–7 element high-entropy alloy compositions are screened for optimal hydrogen adsorption energy (ΔE_H ≈ 0 eV), filtered post-hoc by elemental cost and earth abundance. Composition clusters with predicted activity near the Sabatier optimum are identified.
