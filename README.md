# Environmentally-Conscious ML Screening of the Full Transition Metal Composition Space for HER Electrocatalysts

## Overview
Using a random forest model trained on DFT adsorption energies, a broad sample of 4-6 element high-entropy alloy (HEA) compositions formed from a pool of eighteen transition metals are screened for optimal hydrogen adsorption energy (ΔE_H ≈ 0 eV), filtered post-hoc using AlloySustainability to maximize nine environmental indicators. Composition clusters with predicted activity near the Sabatier optimum are identified.
